This tutorial project contains:

- A PREESM workflow with application flattening, mapping and code generation
- A simple application named TestCom (without hierarchy)
- The architecture model of a PC with 8 cores interconnected by a DMA-like link
- The associated code, IDL with function prototypes and MS Visual solution to execute the code

To launch PREESM on this tutorial:

- Import the tutorial1 project into your Workspace
- Right-click on the workflow. Choose the unique scenario
- Wait for the end of the mapping or stop it from the "Best Latency" editor
- Open the "Executable_Solution.sln" solution in MS Visual
- Launch the execution
